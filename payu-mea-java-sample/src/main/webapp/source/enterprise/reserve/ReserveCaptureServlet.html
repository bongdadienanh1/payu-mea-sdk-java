The APIContext object must be instantiated with a username, password, safekey and mode, the account prefix is optional and is only required if you making use of multiple accounts configuration with sdk_config.properties file.

Payment

A Payment defines the contract of a payment - what is the payment for and who is fulfilling it. Payment is created with a `DoTransaction`

Payment payment = new Payment(); payment.setRequest(doTransaction); DoTransactionResponseMessage doTransactionResponseMessage = null;

Credit a payment by posting to the APIService using a validly configured APIContext object The return object contains the result of the transaction;

try { doTransactionResponseMessage = (DoTransactionResponseMessage) payment.capture(apiContext); LOGGER.info("Captured/finalized payment with id = " + doTransactionResponseMessage.getPayUReference() + " and result code = " + doTransactionResponseMessage.getResultCode()); ResultPrinter.addResult(req, resp, "Capture/Finalize Reserved Payment", Payment.getLastRequest(), Payment.getLastResponse(), null); } catch (PayUSOAPException ex) { ResultPrinter.addResult(req, resp, "Capture/Finalize Reserved Payment", JSONFormatter.toJSON(doTransaction), JSONFormatter.toJSON(doTransactionResponseMessage), e.getMessage()); } return doTransactionResponseMessage; }
